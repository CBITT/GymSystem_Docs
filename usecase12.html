Use Case 12 - Unban Member (Brief & Casual)

Actors: Admin, Gym System

Brief

Admin can unban member and reallow members access to gym system.

Casual

Precondition: User must first be banned

Admin can view a list of all currently banned members and select from the list a particular member that they wish to unban from the gym system.

    System prompts admin "Confirm unban?"
    If admin does not confirm, member remains banned for indefinite amount of time
    If admin confirms, member has access to log back into the gym system as normal
When a member is unbanned, they have full access back into the gym system as before.
