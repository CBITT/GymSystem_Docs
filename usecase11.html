Use Case 11 - Ban Member (Brief & Casual)

Actors: Admin, Gym System

Brief

Admin can ban member from gym system for an indefinite amount of time.

Casual

Admin can view a list of all members and select from the list a particular member that they wish to ban from the gym system.

    System checks with members class if this member is already banned.
    If already banned, prompt "Member is already banned".
    If not already banned, ban member and prompt "Member successfully banned"
When a member is banned they will be unable to login in to the system until they are unbanned. Their account is effectively locked.
