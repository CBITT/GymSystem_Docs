Use Case 6 - Set Workout (Fully Dressed)

Actors: Member, Gym System

Brief

Member can set what type of workout routine they wish to undertake.

Fully Dressed

Precondition: 1) Member must first be logged in. 2)Member must first have a membership. 3)System must scan Book Equipment class to see how many machines are pre booked.

Goals: User is able to select their workout from the provided choices

Business Value: 4

Business Risk: 4

Tasks:

  1. Gym System: The Gym system displays a blank form for the users input, pulls in a list of the available workout options and displays room capacity and current spaces taken.
  2. Member: Selects whether they want to do a weights, cardio or pool workout and starts workout
  3. Gym System: First checks to see if Member has the suitable membership
      4. If the member does, system runs next check.
      If the member does not, system will display workout choices that the member has available to them
  4. Gym System: System checks if member has enough remaining days on membership
      5. If the member does, system runs next check.
      If the user does not, system prompts "Please top up your session count or renew your membership"
  5. Gym System: If the system reports that the gym capacity is full:
      6. prompt user "The gym is full at this time, please come back later" and resets UI
  6. Gym System: Validates users complete input and saves users workout options
